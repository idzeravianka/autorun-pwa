Доступно обновление!

В этом обновлении появится возможность:

  • - Переименовывать сенсоры.
  • - Скрывать сенсоры с рабочего стола.
  • - Добавлена ссылка на телеграм канал.

Для работы с новой функциональностью необходимо нажать на значок редактирования в левом верхнем углу на главной странице приложения, после чего станут доступны для редактирования элементы рабочего стола.

По нажатию на кнопку 2, будет открыто диалоговое окно, в котором можно отредактировать как название выбранного сенсора, так и его отображение на рабочем столе.

Обращаем внимание! Изменения применятся только после того, как кнопка номер 1 будет повторно нажата, по окончанию редактирования.

В меню настроек появилась ссылка на телеграм канал, для обратной связи, коммуникации, оперативной тех.поддержки, предложений по разработке и критике.

Для обновления сейчас нажмите кнопку Обновить (это займет считанные секунды).

По нажатию на кнопку Закрыть приложение будет обновлено в фоне автоматически, при одном из следующих запусков.
