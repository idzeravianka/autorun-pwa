Доступно обновление

Внимание! Доступно обновление в котором:

  • - Появилась возможность переименовывать сенсоры.
  • - Появилась возможность скрывать сенсоры с рабочего стола.

Для работы с новой функциональностью тебе необходимо нажать на значок редактирования в левом верхнем углу на главной странице приложения, после чего тебе станут доступны для редактирования элементы рабочего стола.

По нажатию на кнопку 2, будет открыто диалоговое окно, в котором ты можешь отредактировать как название выбранного сенсора, так и его отображение на рабочем столе.

Обращаю твое внимание на то, что изменения применятся только после того, как ты повторно нажмешь на кнопку номер 1, по окончанию редактирования.

Если ты хочешь обновить приложение сейчас, нажми на кнопку Обновить (это займет считанные секунды).

Если же сейчас нет желания обновляться, можешь нажать на кнопку Закрыть и приложение будет обновлено в фоне автоматически, при одном из следующих запусков.
